Smart Campus Operations Hub – IT3030 PAF Assignment 2026

This project is a full-stack web-based Smart Campus Management System developed using Spring Boot REST API and React.js.

The system enables university students and staff to manage:

• Facility and asset catalogue (rooms, labs, equipment)
• Resource booking with approval workflow and conflict management
• Maintenance and incident ticketing system with image attachments
• Real-time notifications
• Secure authentication and role-based authorization (OAuth 2.0)

This project is developed as part of the Programming Applications and Frameworks (IT3030) module at SLIIT.
